# Отчет по практической работе №5

## Выполнил: Золотых Владимир

## Тема
Основы работы с Git

## Цель
Изучить систему контроля версий Git

## Команды

### Инициализация репозитория
```
git init
```
Создаёт новый репозиторий

### Добавление файлов
```
git add .
```
Добавляет файлы в отслеживание

### Коммит
```
git commit -m "сообщение"
```
Сохраняет изменения
